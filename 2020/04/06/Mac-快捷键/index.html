Mac 常用快捷键

2020-04-06 Essays

文章目录

  1. 截屏
  2. 应用
  3. 文本
  4. 文件
  5. 浏览器

截屏

  • command + shift + 3 ：全屏截屏
  • command + shift + 4 ：选定区域截屏

应用

  • command + tab ：应用间切换
  • command + shift + u ：使用工具

文本

  • control + a : 移动至行首
  • control + e : 移动至行尾

文件

  • command + 0 : 打开文件
  • command + option + l : 打开下载
  • command + c : 复制
  • command + v : 粘贴
  • command + alt + v : 移动
  • command + alt + ← ：删除文件（Tab键确认、空格键确认删除）

浏览器

  • command + shift + { ：上个标签页
  • command + shift + } ：下个标签页
  • command + alt + ← ：上一个标签
  • command + n ：新开一个标签
  • command + w : 关闭当前标签
